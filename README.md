## Installation.

- In the terminal you can clone the repository:
  `git clone https://github.com/JP62442/pruebaTecnicaPosts.git`

- Enter the folder that is created:
  `cd pruebaTecnicaPosts `

- Install the dependencies:
  `npm install`

- To execute:
  `npm run dev`
